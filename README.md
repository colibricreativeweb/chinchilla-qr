## :gear: Install & Run

You'll need at least `node 16` to compile everything:

```bash
# Install dependencies
npm install

# Run the application
npm run dev
```

## :star2: Main features

**Frontend / UI:**
* React
* Tailwind CSS
* QR Code generation using [qr-image](https://www.npmjs.com/package/qr-image)
